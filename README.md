# Antonio Figueroa Ontology (Lite)

A curated public reference ontology spanning forensic accounting, integrated data science methods for forensic and valuation work, and corporate financial valuation (with particular emphasis on intangible asset valuation as it relates to consequential business income losses in insurance contexts). This repository is the public lite version of a larger private personal ontology and is published for two purposes: (1) to provide a structured reference and onboarding aid for junior analysts working with me on forensic and valuation engagements, and (2) to serve as a publicly demonstrable artifact of the multi-disciplinary work I have developed over fifteen plus years of practice.

## Author

Antonio Figueroa

## Repository Scope

This repository contains a representative subset of materials from a larger private ontology, including:

| Category | Description |
|----------|-------------|
| Markdown context files | Domain reference notes, terminology guides, methodology summaries |
| Prompt templates and Skills | Reusable Claude and GPT Skills for forensic accounting and data science workflows |
| Agent definitions | Configuration for agentic workflows applicable to claims, valuation, and litigation support tasks |
| MCP server source code | Selected Model Context Protocol servers tailored to forensic and analytical use cases |
| Example cases | Anonymized illustrative examples used to demonstrate methodology |

The materials selected for inclusion in this lite version represent the components most useful for educational and reference purposes. The full private ontology contains additional proprietary materials, working files, and engagement-specific content that are not appropriate for public distribution.
